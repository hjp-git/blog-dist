ES6 Map对象

# 简介

ES6 提供了 Map 数据结构。它类似于对象，也是键值对的集合。
但是 key 的范围不限于字符串，各种类型的值（包括对象）都可以当作 key 。
Map 对象保存键值对，并且能够记住键的原始插入顺序。
Map 也实现了 iterator 接口，所以可以使用 ...... 和 for…of 进行遍历。

# Map 对象和 Object 的区别

  • 一个 Object 的键只能是字符串或者 Symbol ,但一个 Map 的键可以是任意值
  • Map 中的键值是有序的，而添加到对象中的键值则不是
  • Map 的键值对个数可以从 size 属性获取，而 Object 的键值对个数只能手动计算
  • Object 都有自己的原型，原型链上的键名有可能和你自己在对象上的设置的键名产生冲突

# 声明Map对象

let myMap = new Map();
console.log(myMap);//Map(0) {}
console.log(myMap.size);//Map对象的长度
  • key是字符串
let myMap = new Map();
let keyStr = "I am string";
myMap.set(keyStr, '和字符串关联的值');
console.log(myMap);
  • key是对象
let myMap = new Map();
let keyObj = {
    name: 'My name is yasuo'
};
myMap.set(keyObj, '和对象关联的值');
console.log(myMap);
  • key是函数
let myMap = new Map();
let keyFun = function() {};
myMap.set(keyFun, '和函数关联的值');
console.log(myMap);

# Map 相关的函数与属性

MapObject.clear() – 移除 Map 对象的所有键/值对 。 MapObject.set() – 设置键值对，返回该 Map 对象。
MapObject.get() – 返回键对应的值，如果不存在，则返回 undefined 。
MapObject.has() – 返回一个布尔值，用于判断 Map 中是否包含键对应的值。
MapObject.delete() – 删除 Map 中的元素，删除成功返回 true ，失败返回 false 。
MapObject.size – 返回 Map 对象键/值对的数量。
MapObject.keys() - 返回一个 Iterator 对象， 包含了 Map 对象中每个元素的键 。
MapObject.values() – 返回一个新的 Iterator 对象，包含了 Map 对象中每个元素的值 。

# Map的迭代

  • for...in 遍历 Object ，不能遍历 Map对象
  • for...of 遍历 Map 对象，不能遍历 Object
  • forEach() 遍历 Map 对象或者数组对象，不能直接遍历 Object
for (let key in myMap) {
  // 无输出
  console.log(key); 
}

for (let key of myMap) {
  // 有输出，每一条都是键值对（放在数组在中，第一个值是属性名，第二个值是属性值）
  console.log(key); 
}

myMap.forEach((value, key) => {
  console.log(key + "=>" + value);//直接输出key和value
});

# 数组转Map对象

数组转 Map 对象,前提条件是二维数组，关联二维数组

let arr = [
  [1, "a"],
  [2, "b"],
  [3, "c"]
];
let myMap = new Map(arr);
console.log(myMap);

# Map对象转数组

Array.from(MapObject)

let arr = Array.from(myMap);
console.log(arr);

# 克隆Map对象

// 定义Map对象
let myMap = new Map();
myMap.set('name', 'zhangsan');
myMap.set('age', 20);

//Map对象构造函数生成实例，迭代出新的对象
let myMap1 = new Map(myMap);

console.log(myMap);//{"name" => "zhangsan", "age" => 20}
console.log(myMap1);//{"name" => "zhangsan", "age" => 20}
console.log(myMap == myMap1);//false
console.log(myMap === myMap1);//false

# 合并Map对象

let myMap = new Map();
myMap.set('name', 'zhangsan');
myMap.set('age', 20);

let myMap2 = new Map();
myMap2.set('name1', 'zhangsan');
myMap2.set('age1', 20);

let myMap3 = new Map([...myMap, ...myMap2]);
console.log(myMap3);

合并 Map 对象时，如果有重复的键，则后面的会覆盖前面的键值
