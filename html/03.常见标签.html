# href 属性

<!--默认打开的还是当前页面，会刷新一下重新打开，会回到页面顶部。效果相当于F5后重新加载了页面。-->
<a href =""></a> 

<!--浏览器地址栏网址后面会多1个#。不会刷新页面，会回到页面顶部。只是在页面内部跳转，并没有重新加载页面。-->
<a href ="#"></a> 

<!--执行一段空白的baijavascript语句，返回空或者false值，从而防止链接跳转-->
<a href ="javascript:;"></a>

<!--在本身这个网页窗口打开新的网页链接-->
<a href="https://www.jd.com/" target="_self"></a>

# table,tr,td 的属性

border 边框
width  宽度
height  高度
bgcolor  背景颜色
align  水平排列方式（默认left）
valign 单元格垂直排列方式（默认middle）
cellspacing  单元格外间距 	默认为2个像素 （属性写在table里）
cellpadding  单元格内间距	默认为1个像素 （属性写在table里）
rowspan     跨行合并    行和行之间的合并 从上往下计算（属性写在td里）
colspan     跨列合并	  列和列之间的合并 从左往右计算（属性写在td里）

table.style ==> border-spacing 设置单元格之间的间距

# form 两个默认的属性

action 访问后端的地址,默认不写是跳转到当前页面

method 发送数据的方式

  • get 一般是从服务器端获取数据,明文提交会在地址栏显示提交的内容,包括密码隐私性安全性交叉较差,提交的数据的大小是受限制 kb级别
  • post 一般是从服务器端获取数据,隐式提交 在地址栏看不到提交的内容,传输没有数据大小限制

# 音频audio

<audio src="urlfile/BC221 - 如果你能感同我的身受.mp3" controls="controls"></audio>
1

（谷歌浏览器把autoplay属性禁用了）

属性 值 描述
autoplay autoplay 音频就绪后播放
controls controls 向用户显示控件（播放按钮）
loop loop 循环播放
src url 音频文件路径

# 视频video

<video src="urlfile/video.mp4" controls></video>
1

（谷歌浏览器把autoplay属性禁用了）但是可以通过添加muted来解决播放问题

属性 值 描述
autoplay autoplay 视频就绪后播放
controls controls 向用户显示控件（播放按钮）
width 像素值 设置播放器宽度
height 像素值 设置播放器高度
loop loop 循环播放
preload auto(预先加载视
none(不预先加载)		 规定是否预加载视频，有autoplay就忽略该属性
src url 视频文件路径
poster imgurl 加载等待的画面图片
muted muted 静音播放
