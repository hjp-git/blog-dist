jQuery使用

# jQuery引入

要使用jQuery，必选引入jQuery
jQuery链接 (opens new window)

建议在body底部引入jQ,防止jQ报错,阻塞页面渲染

第一种方式，引入网络 jQuery CDN

<script src="https://cdn.bootcdn.net/ajax/libs/jquery/3.5.1/jquery.js"></script>

第二种方式，下载本地引入

<script src="jq.js"></script>

# jQuery入口函数

如果发现控制台出现 $ is not defined
第一种原因：没有引入JQ
第二种原因：引入了JQ，但是放置的位置顺序不对

//原生js，入口函数。页面上所有内容加载完毕，会执行。

// 1.第一种写法    window,document,this 这三个不需要加引号
$(document).ready(function(){
   // code....
    console.log($('div'));

})

// 2.第二种写法 上面方法的简写
$(function(){
 console.log(11111);

 })

  1. 等着 DOM 结构渲染完毕之后再执行内部的代码,不必等着所有外部资源加载完成,jQ帮我们完成了封装
  2. 相当于原生js中的 window.onload 事件
  3. 不同于原生js中的 onload 事件等页面文档 外部的js,css文件,图片等加载完毕才执行内部代码,一个页面中只能有一个 window.onload 事件,而jQ可以有多个jQ的入口函数
