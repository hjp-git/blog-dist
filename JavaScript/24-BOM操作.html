BOM操作

BOM的定义
Browser Object Model 即 浏览器对象模型 。浏览器页面初始化时，会在内存创建一个全局对象，用来描述当前窗口的属性和状态，这个全局对象被称为浏览器对象模型

DOM 的标准化组织是 W3C， JavaScript 的语法标准化组织是 ECMA ，但是 BOM 没有官方标准，它最初是 Netscape 浏览器标准一部分，也就是说，对现代浏览器，每个浏览器都有自己的BOM实现方法，所以直接使用 BOM 会有兼容性问题

BOM包含哪些内容

BOM有一个核心对象 window，window对象包含了6大核心

  • document 文档对象
  • frames HTML框架(基本淘汰)
  • history 页面的历史记录
  • location 页面的地址
  • navigator 浏览器的相关信息
  • screen 用户显示屏幕相关信息

# window 对象

BOM的核心对象就是 window，window 对象也是 BOM 的顶级对象，所有浏览器都支持 window 对象，它代表的是浏览器的窗口

JavaScript的所有 全局对象、全局方法和全局变量 全都自动归为 window对象 的方法和属性， 在调用这些方法和属性可以省略 window。DOM也 是window对象的属性

window.innerWidth	// 返回浏览器可视窗口的宽度(包括滚动条宽度)
window.innerHeight	// 返回浏览器可视窗口的高度(包括滚动条高度)

window.outerWidth	// 返回浏览器窗口的宽度(包括滚动条宽度)
window.outerHeight	// 返回浏览器窗口的高度(包括滚动条高度及浏览器工具栏的高度)

// 获取可视化区域的宽高，以上四个window的属性对于IE9以下浏览器不支持，需要写兼容写法

var clientWidth = document.documentElement.clientWidth || document.body.clientWidth || window.innerWidth
var clientHeight = document.documentElement.clientHeight || document.body.clientHeight || window.innerHeight

# window 对象的方法

window.open(url,name,specs)

URL 新窗口需要载入的 url 地址。 可以是 web 上的 html 页面也可以是图片文件或者其他任何浏览器支持的文件格式。
name 指定在某个窗口打开 _blank新窗口、 _self原窗口、 _parent父窗口
specs 规定打开窗口的样式（窗口的大小，样式之间用逗号连接，不用空格），是否在history列表中替代当前页面

window.open('http://www.jd.com/','_blank','width = 400px,height = 200px')

window.close()  // 关闭当前窗口
window.moveTo() // 移动窗口到特定的位置
window.resizeTo()  // 修改窗口大小

# window 三个弹窗

alert -- 警告框（当你需要确保某个信息传递给用户时，使用 Alert box，用户需要点击“确定”来继续下一个步骤，没有返回值 undefined ）

confirm -- 确认框（当你需要用户验证或者接受某个信息时，使用 Confirm box，用户需要点击“确定”或者“取消”来继续下一个步骤，点击“确定”，返回true，点击“取消”，返回 false ）

prompt -- 提示框（当你希望用户进入页面前，先输入某个信息时，使用 Prompt box，用户需要输入一个值，然后点击 “确定” 或者 “取消” 来继续下一个步骤，如果用户点击“确定”，返回用户输入的值（永远都是字符串类型），如果用户点击“取消”，返回 null ）

var btn = document.querySelectorAll('button');
var myPage;

btn[0].onclick = function(){
  myPage = window.open('','','width = 400px,height = 200px',);
  myPage.document.write("<p>这是我的窗口</p>");
}

btn[1].onclick = function(){
  myPage.moveTo(400,300);
  myPage.focus();
}


btn[2].onclick = function(){
  myPage.resizeTo(500,500);
  myPage.focus();
}

btn[3].onclick = function(){
  myPage.close();
}

# document 对象

渲染引擎在解析HTML对象 时，会生成一个document对象 ，即文档对象。

它是HTML文档的根节点，每一个元素都会生成对应的DOM对象，由于元素之间的层级关系，整个HTML代码解析结束之后，会生成一歌不同节点组成的树形结构，成DOM树

document 用于描述DOM树的状态和属性，并提供很多操作 DOM 的 API

# frames 对象

frames 指 HTML 子框架，即在浏览器里嵌入另一个窗口，父框架和子框架拥有独立的作用域和上下文

元素 frame 在 HTML5 中已废弃

# history 对象

history 对象包含浏览器的历史记录，这些历史记录以栈的形式保存，页面前进则入栈，页面返回则出栈

它是一个只读对象，为了保护用户的隐私，JavaScript在获取这个对象时会受到限制，history对象不允许未授权代码访问会话历史中的其他页面地址，但可以到货到其他会话历史指向的页面。

未授权代码无法清除history，也不能禁止后退/前进的功能

# history 对象的方法

history.back() 跳转到历史记录中的前一个链接 -- 回退
history.forward() 跳转到历史记录中的后一个链接 -- 前进
history.go() 上面两个方法的集合

history.go(0)     // 表示刷新当前页面
history.go(-1)    // 表示跳转到历史记录中的前一个链接 --  回退  相当于back()    
history.go(1)     // 表示跳转到历史记录中的后一个链接 --  前进  相当于forward()

# location 对象

location 对象可以获取当前页面地址以及重定向到一个新页面

# location 对象的属性

window.location.href 返回当前页面的地址,当前页面打开地址
window.location.hash 返回当前页面的hash值(#后面的字符串)
window.location.hostname 返回当前页面的域名
window.location.pathname 返回当前页面的路径和文件名
window.location.protocol 返回网页使用的协议（http: 还是 https: ）
window.location.port 返回当前页面的端口，如果页面使用的是默认端口（http:80,https:443） 则大多数浏览器会显示为0或者不显示
window.location.assgin 加载一个新的文档

# location 对象的方法

window.location.reload() 重载页面 -- 刷新页面
window.location.replace() 重定向页面 -- 跳转到新的页面

# navigator 对象

navigator 对象是一个只读对象，它是描述浏览器本身的信息，包括浏览器的名称、版本、语言、系统平台、用户特性字符串等信息

navigator 对象有很多获得浏览器信息的方法，但是大多数不提倡使用，因为他们返回的数据并不准确 (如 appCodeName,appName,platform 等)

# navigator 对象的属性

navigator.appName 返回浏览器名称(IE11, chrome, firefox, safari 都返回 Netscape )
navigator.appCodeName 返回当前浏览器的内核(chrome, IE, safari 和 Opera 都返回 Mozilla )
navigator.userAgent 返回由客户机发送服务器的 user-agent 头部的值

function getBroswer(){
    // 1.通过navigator.userAgent获取浏览器信息  ---  字符串
    var userAgent = navigator.userAgent;        
    // 2.通过字符串的方法查找对应的浏览器的名称Chrome  Firefox  Safari
    // 3.if判断是哪个浏览器 
    if (userAgent.indexOf('Chrome') != -1) {
    	return 'Chrome浏览器';
    } else if (userAgent.indexOf('Firefox') != -1) {
    	return 'Firefox浏览器';
    } else if (userAgent.indexOf('Safari') != -1){
    	return 'Safari浏览器';
    } else if (userAgent.indexOf('OPR') != -1 || userAgent.indexOf('Opera') != -1){
    	return 'Opera浏览器';
    }  else{
    	return 'IE浏览器';
    }         
}
console.log(getBroswer());

# screen 对象

screen 对象提供了用户显示屏幕的相关属性，比如显示屏的宽度，高度，可用宽度，可用高度

# screen 对象的属性

screen.width 显示屏的宽度 screen.height 显示屏的高度
screen.availWidth 屏幕的可用宽度 screen.availHeight 屏幕的可用高度
