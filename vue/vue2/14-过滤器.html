过滤器

  • Vue.js 允许你自定义过滤器，可被用于一些常见的文本格式化。
  • 过滤器可以用在两个地方：双花括号插值和 v-bind 表达式 (后者从 2.1.0+ 开始支持)。过滤器应该被添加在 JavaScript 表达式的尾部，由 | 符号指示
  • 过滤器不改变真正的 data，而是改变渲染的结果，并返回过滤后的版本

  • 过滤器全局注册是 filter ，而局部注册是 filters
  • 当全局过滤器和局部过滤器重名时，会采用局部的过滤器

# 全局过滤器

  • 全局过滤器注册写在 vue 实例之前
<div id="app">
  {{msg}}
  <!--这里传参 -->
  <hr> {{msg | lower(5,"dayu","小于等于")}}
</div>

//注册全局过滤器
// Vue.filter('名字',函数(val));
Vue.filter("lower", function(val, n, a, b) {
  // 1.第一个参数，处理的值,不需要传值，这个值从data里获取
  // 2.从第二个参数往后，表示传递的参数
  if (n > 5) {
    return val + a;
  } else {
    return val + b;
  }
});

# 局部过滤器

  • 局部注册过滤器，只在当前的Vue实例中有效
<div id="app">
  输入字符串：<input type="text" v-model='msg'>
  <hr>
  <h2>data中正常：{{msg}}</h2>
  <hr>
  <h2>过滤后渲染：{{msg | haowan(15,'帅','无聊')}}</h2>
</div>

var vm = new Vue({
  el: "#app",
  data: {
    msg: "哈哈，你好"
  },
  methods: {

  },
  // 局部注册过滤器
  filters: {
    haowan: function(val, n, a, b) {
      // filter参数
      // 1.第一个参数 表示要处理的值
      // 2.从第二个参数往后，表示传递的普通字符串
      if (n < 10) {
        return val + a;
      } else {
        return val + b;
      };
    }
  }
});
