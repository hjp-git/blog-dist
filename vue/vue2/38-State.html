State

# 基本使用

  • 由于 Vuex 的状态存储是响应式的,从 store 实例中读取状态最简单的方法就是在计算属性中返回某个状态
  • 获取 Vue x里面的状态 vm.$store.state.变量名

@/store/index






 
 
 


import Vue from 'vue'
import Vuex from 'vuex'
Vue.use(Vuex)

export default new Vuex.Store({
  state: {
    num: 0
  },
})

组件中使用



 








 





<template>
  <div>
    <h1>{{$store.state.num}}</h1>
    <h1>{{num}}</h1>
  </div>
</template>

<script>
export default {
  computed: {
    num() {
      return this.$store.state.num
    }
  },
}
</script>

# mapState 辅助函数

  • 当一个组件需要获取多个状态的时候，将这些状态都声明为计算属性会有些重复和冗余。为了解决这个问题，我们可以使用 mapState 辅助函数帮助我们生成计算属性

这种方式是静态的,无法映射到 v-model ,需要手动更新 state 数据

@/store/index

import Vue from 'vue'
import Vuex from 'vuex'
Vue.use(Vuex)

export default new Vuex.Store({
  state: {
    num:0,
    num1:1,
    num2:2,
  },
  mutations: {
    set_num(state,val){
      state.num = val
    },
  },
})

# 对象形式

  • 一般以对象形式映射










 

 
 
 
 
 
 
 
 
 













<template>
  <div class="home">
    <input type="number" :value="num" @input="changeVal" />
    <h4>{{ num }}</h4>
    <h4>{{ num1 }}</h4>
    <h4>{{ num2 }}</h4>
  </div>
</template>

<script>
import { mapState } from "vuex";
export default {
  computed: mapState({
    num: state => state.num,
    //符串参数 num' 等同于 `state => state.num
    num1: "num",
    // 为了能够使用 `this` 获取局部状态，必须使用常规函数
    num2: function (state) {
      return state.num + this.str;
    },
  }),
  data(){
    return {
      str: "只王八"
    }
  },
  methods: {
    changeVal(val) {
      this.$store.commit("set_num", val.target.value);
    },
  },
};
</script>

# 数组形式

  • 当映射的计算属性的名称与 state 的子节点名称相同时,我们也可以给 mapState 传一个字符串数组,








 


 













<template>
  <div class="home">
    <input type="text" :value="num" @input="changeVal" />
    <h4>{{ num }}</h4>
  </div>
</template>

<script>
import { mapState } from "vuex";
export default {
  // 映射 this.num 为 $tore.state.num
  computed:mapState(['num']),
  data() {
    return {
      str: "呼呼",
    };
  },
  methods: {
    changeVal(val) {
      this.$store.commit("set_num", val.target.value);
    },
  },
};
</script>

# 计算属性混合

mapState 函数返回的是一个对象。我们可以将它与局部计算属性混合使用











 



 
 
 
 
 
 
 














<template>
  <div class="home">
    <input type="text" :value="num" @input="changeVal" />
    <h4>{{ num }}</h4>
    <h4>{{ num1 }}</h4>
    <h4>{{ num2 }}</h4>
  </div>
</template>

<script>
import { mapState } from "vuex";
export default {
  computed: {
    // 使用对象展开运算符将此对象混入到外部对象中
    ...mapState({
      num: (state) => state.num,
      num1: "num1",
      num2: function (state) {
        return state.num2 + this.str;
      },
    }),
  },
  data() {
    return {
      str: "只王八",
    };
  },
  methods: {
    changeVal(val) {
      this.$store.commit("set_num", val.target.value);
    },
  },
};
</script>
