# 选择器

选择器 用来匹配页面中的元素

/*通用选择器  匹配页面中所有的元素  一般用于项目的初始化代码  支持多个*/
*{
    font-size: 20px;
}

/* 元素/标签/标记选择器   匹配页面中  所有指定的元素  支持多个 */
 p{
     color:red;
 }

/*id选择器 :  语法 #id值      匹配页面中指定id值的元素  唯一*/
#man{
    background: #f3ac32;
}

/*类选择器  语法 .类名      匹配页面中指定类名的元素  支持多个*/
.man{
    color:yellow;
}

<!--  一个元素 支持多个 类的划分 中间用空格可开 -->
<!-- 支持多类 的写法 -->
<div class="font_color bgcolor">
    男人何必为难男人
</div>

# 选择器之间的具备优先级问题

权重 值越大 优先级越高

#id -- 100
.class -- 10
元素选择器 -- 1
通配符 -- 0
继承的样式没有权值。-- 空

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
    <style>
      /* 元素/标签/标记选择器   匹配页面中  所有指定的元素  支持多个 */
      p {
        color: red;
      }
      /*通用选择器  匹配页面中所有的元素  一般用于项目的初始化代码  支持多个*/
      * {
        font-size: 20px;
      }
      /*id选择器 :  语法 #id值      匹配页面中指定id值的元素  唯一*/
      /*权重100*/
      #man {
        background: #f3ac32;
      }
      /*类选择器  语法 .类名      匹配页面中指定类名的元素  支持多个*/
      .font_color {
        color: yellow;
      }
      /* 找一个 必须是div  必须是有一个类叫做bgcolor的人*/
      /*权重 1+10=11    */
      div.bgcolor {
        background: red;
      }
    </style>
  </head>
  <body>
    <p>离离原上草</p>
    <p>一岁一枯荣</p>
    <p>野火烧不尽</p>
    <!-- 不符合条件-->
    <p class="bgcolor">春风吹又生</p>
    <div>曾经沧海难为水</div>
    <!-- 显示橘黄色的原因是 #的权重大于 .类的权重-->
    <div id="man" class="font_color bgcolor">男人等于难人</div>
    <!--  支持多类 的写法 -->
    <div class="font_color bgcolor">男人何必为难男人</div>
  </body>
</html>

# 后代选择器

通过一个空格表示后代关系，通过父级标签找到它的后代标签

.box span{}	表示选中.box里面的span标签

# 子元素选择器

通过把父级标签写在前面，子级标签写在后面，中间跟一个 > 进行连接
这里的子指的是亲儿子，不包括孙子、重孙之类

div>h3{} 表示选中div里下一级的h3

# 并集选择器

多个选择器用逗号隔开表示同时选中，作用是一起书写样式

h1,p,ol,li,.box span{background:red;}

# 交集选择器

通过不同选择器连续书写，表示指定选择，作用是选中指定的标签

p.box{}表示选中带有box类名的p标签

# 高级选择器

# 兄弟选择器

相邻兄弟 查找下一个兄弟元素

选择器1+选择器2{}

通用兄弟 查找当前元素之后的所有兄弟

选择器1~选择器2{}

/*  相邻兄弟选择器   语法 + */
.san+li+li{
    color:red;
}
/*通用兄弟:  匹配当前元素之外的后面的所有的兄弟*/
.san~li{
    font-size: 30px;
}
/*兄弟元素  可以写空格 还是表示兄弟关系 */
.san+li + li{
    color:red;
}

# 属性选择器

.box[class]			  选择具有class属性的元素.box

.box[class="red"]     选择具有属性class="red"的元素.box

.box[class^="a"]	  选择具有class属性，且属性值以a开头的元素.box

.box[class$="1"]	  选择具有class属性，且属性值以1结尾的元素.box

.box[class\*="m"]	  选择具有class属性，且属性值中含有m的元素.box

支持自定义属性,严格到多一个空格都会匹配不到

# 伪类选择器

actions伪类 ==> 有 actions 行为

选择器:disabled{}  	元素禁用时

选择器:link{}	   	表示元素默认样式（无操作样式）

选择器:visited{}     超链接访问过后的样式

选择器:hover{}	   鼠标移入元素时，元素的样式

选择器:active{}	   鼠标点击元素时，元素的样式		

选择器:focus{}	  	超链接获取焦点时的样式

选择器:checked{}   	元素被勾选时的样式

选择器:target{} 	跳转到目标元素时样式

<head> 
  <style>
    /*鼠标没有点击之前的样式*/
    a:link{
      color:green;
    }
    /*鼠标移入的效果*/
    a:hover{
      color:yellow;
    }
    /*点击不放的样式*/
    a:active{
      color: red;
    }
    /*已经访问过的样式*/
    a:visited{
      color:pink;
    }
    /*鼠标获取焦点时的样式*/
    input:focus{
      border:1px solid red; 
    }
  </style>
</head>
<body>
  <a href="http://www.baidu.com">度娘</a>
  <input type="text">
</body>

否定伪类

:not(:选择器){}  排除

<head>
  <style>
    /*第一个元素不变 其他人都变   可以通过否定*/
    ul li:not(:first-child){
      background-color: red;
    }
    /* 第一个 最后一个元素不变  多个冒号多重否定 */
    ul li:not(:first-child):not(:last-child){
      color:yellow;
    }
  </style>
</head>
<body>
  <ul>
      <li>大张伟</li>
      <li >张三丰</li>
      <li>张翠山</li>
      <li>张无忌</li>
      <li>张发贵</li>
      <li>张志强</li>
      <li>张三</li>
  </ul>
</body>

结构伪类 ==> 通过元素的结构去查找

指定类型:first-child	  父级的第一个指定类型子级

指定类型:last-child 	    父级的最后一个指定类型子级

指定类型:nth-child()	   父级的任意一个指定类型子级

指定类型:nth-last-child(n)  父级的倒数第n个指定类型的子级

指定类型:nth-child(2n)      支持数学表达式  一般用来做隔行效果

指定类型:first-of-type  		父级的第一个指定类型子级

first-of-type 是找到的第一个
first-child 必须是第一个

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <style>
    p:first-of-type{
      color: blue;
    }
    span:first-child{
      color: red;
    }
  </style>
</head>
<body>
  <div>
    <span>00000</span>
    <p>111111</p>
    <p>2222</p>
    <div>
      <p>aaa</p>
      <p>bbb</p>
      <span>ccccc</span>
    </div>
  </div>
</body>
</html>

# 伪元素选择器

创建出来的元素在 Dom 中查找不到，所以称为伪元素
伪元素和标签选择器一样，权重为 1

常用于内容生成(非常重要)
::before 在元素内部的前面生成内容
::after 在元素内部的后面生成内容

before 和 after 必须有 content 属性并给空值，表示插入的内容
before 和 after 创建的是一个元素，但是属于行内元素

<head>
	<style>
    .parent{
      width: 400px;
      height: 400px;
      background-color: red;
    }
    .son{
      width: 100px;
      height: 100px;
      background-color: #0c0c0c;
      /*默认情况父元素没有边框给子元素设置上外边距会作用到父元素身上*/
      margin-top: 50px;
    }
    /*内容生成 解决*/
    .parent:before{
      content: "";
      display: table;
    }
  </style>
</head>
<body>
   <div class="parent">
       <div class="son"></div>
   </div>
</body>

<head>
  <style>
    .parent {
      background-color: red;
    }
    .son {
      width: 100px;
      height: 100px;
      background-color: #0c0c0c;
      /*默认情况父元素没有设置高度 子元素浮动父元素高度为 0  */
      float: left;
    }
    /*通过内容生成解决*/
    .parent:after {
      content: "";
      display: block;
      clear: both;
    }
  </style>
</head>
<body>
  <div class="parent">
    <div class="son"></div>
  </div>
</body>
