2D转换

# 位移

原有的位置保留 并且脱离原有的平面
只沿着x轴和y轴移动，不会对其他元素造成影响，对行内标签无效

transform: translate(xpx, ypx);
transform: translateX(xpx);
transform: translateY(ypx);
transform: translate(50%,50%);  /*里面的xy值的百分比值是相对于自身宽高的百分比的*/

# 旋转

transform: rotate(度数值deg);

/* 度数值为正，顺时针，度数值为负，逆时针。 */
/* 默认以选中元素的中心为旋转的中心点(50% 50%)（z轴） */

x 轴：水平向右 -- x 轴右边是正值，左边是负值
y 轴：垂直向下 -- y 轴下面是正值，上面是负值
z 轴：垂直屏幕 -- 往外边的是正值，往里面的是负值

/* x轴 水平方向  烤羊腿  爆米花*/
transform: rotateX(720deg);
/* y轴  垂直方向    旋转门 旋转木马*/
transform: rotateY(720deg);
/* z轴  从电脑屏幕 到人眼的方向  电风扇 摩天轮*/
transform: rotateZ(720deg);

# 中心点

中心点可以设置旋转的，也可以设置缩放的

transform-origin: x y z;
transform-origin:50% 50% 0; /* 默认值 */

transform-origin 属性值可以是 百分比、em、px等具体的值，也可以是top、right、bottom、left 和 center 这样的关键词

# 缩放

transform: scale(x,y);

里面的值是数字，倍数，不跟单位，默认一倍不变化,单值就是等比例缩放，大于1就是放大，小于1就是缩小
优点：不会影响其他盒子，而且可以设置缩放的中心点

通过 缩小元素的大小 可以改变 字体大小 默认PC 浏览器的支持的最小的字体是 12px

# 水平和垂直的倾斜

transform: skew(ydeg,xdeg);

/* y正值，向上左倾斜 */
/* x正值，向右下倾斜 */

# 2D简写

transform: translate(400px, 0) rotate(90deg)....;

属性的顺序会改变转换的结果
当我们同时有位移和其他属性的时候，要把位移放到最前面（2D）
